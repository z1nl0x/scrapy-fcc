- Creating the project with scrapy: scrapy startproject bookscraper
- Creating the spider that will be responsible for crawling the data: scrapy genspider bookspider books.toscrape.com 
- Install ipython to show a verbose shell execution with the command: pip install ipython
- Edit the scrapy.cfg file adding the shell used
- Execute the command to run: scrapy shell
- To execute the spider use scrapy crawl name-of-thespider, ex: scrapy crawl bookspider
- To export data into an specific type of file ex (CSV): scrapy crawl bookspider -O bookdata.csv 
- To specify the exported file generated by the spider set this on settings.py like this:
        FEEDS = {
            'booksdata.json' : {'format': 'json'}
        }
